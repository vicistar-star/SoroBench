# Security Policy

## Supported Versions

Currently, we support the following versions of SoroBench:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

We take the security of SoroBench seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not** open a public GitHub issue.
2. Send an email to the maintainers (contact info in the main repository).
3. Provide a detailed description of the vulnerability and steps to reproduce it.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary. We appreciate your help in keeping SoroBench secure for everyone.
